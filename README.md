# 咕咕导航

基于 [Arco Design Vue](https://arco.design/vue/component/icon) 写的自用导航。

可以直接在线编辑导航内容，内容通过 `localStorage` 以json形式的配置保存在 `customBox` 中。

在线示例： https://gugu-nav.vercel.app/
